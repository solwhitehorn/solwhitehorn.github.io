### Metallic Child Demo running on Linux

---

I was not looking for this game but I found while browsing rogue-lite games on [Steam](https://store.steampowered.com/app/1490610/METALLIC_CHILD/) and was kinda interested in testing it on Linux since it has a demo available. 

The game defines itself as a rogue-lite action core-action game where you take the control of an android left alone to battle against a rebellion of robots on a space station set to crash on earth. That's quite the setup for this very anime styled game that takes all of the roguelike formula.
